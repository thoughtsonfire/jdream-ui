# 安装

## npm 安装
推荐使用npm安装
```
npm i jdream-ui  -S
```
## 引入
### 整体引入
在main.js文件中
```js
import jdreamUI from 'jdream-ui'
import 'jdream/lib/index.css'
Vue.use(jdreamUI)
```
### 部分引入
在main.js文件中
```js
import {hxButton} from 'jdream-ui'
import 'jdream/lib/index.css'
Vue.use(hxButton)
```
